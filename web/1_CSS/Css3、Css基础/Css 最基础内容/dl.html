主要演员：刘德华，周润发
影片长度：90分钟
内容介绍：我是影片内容，你可能听过这样一种说法：“狗狗只能看到黑和白”，或者“狗只看到灰色”。当然，这些说法都不正确，但是这提醒我们认清楚一点：动物眼中的色彩世界与人类的大不相同，甚至于超出我们的理解范围。 首先，我们先来简单了解下人类眼睛工作的原理：作为视觉器官，眼睛从感应明暗到提供视觉，是最重要的感官之一。为了形成视觉，光线经过折射到达对光敏感的视网膜。视网膜上的感光细胞（锥细胞和杆细胞）会将光信号传递给大脑，形成图像。 基本上，视杆细胞负责感应光线，视锥细胞负责捕捉颜色。所以色盲的动物无疑是缺乏视锥细胞，而只能在白天活动的动物是因为没有视杆细胞，但是这两种极端很少见，大多数动物都含有这两种细胞。 那么，动物眼里的世界到底是怎样的呢？ 1、 猫和狗能看到的颜色很少，但是在黑暗中，它们的视力却比人类厉害 人类眼睛具有3种视锥细胞，分别负责长、中、短三种波长范围的颜色。而狗的视锥细胞却只有两种类型，缺少接受红色的视锥细胞。所以，狗狗能够看到的颜色不如我们多，但是他们并不属于色盲。猫咪也是如此，世界上大多数哺乳动物都是这样。 但是，猫和狗的视杆细胞却远远多于我们拥有的，这意味着，在黑暗中，它们的视力优于我们。所以，人类并不处于视觉链的顶端。 2、 蜜蜂和蝴蝶能够看到紫外线 我们倾向于认为，因为拥有3种类型的视锥细胞，所以我们能够看到所有应该被看到的颜色。但是事实并非如此。很多其他动物有更多的视锥细胞，能够看到更多的颜色，例如，蜜蜂和蝴蝶。 研究表明，蜜蜂和蝴蝶有4种视锥细胞类型，它们能够看到包括紫外线在内的神奇光谱。当然，它们看不出太多的细节。使这一切成为可能的光感受细胞很重要，因为借助它们，昆虫能够以紫外线模式分辨许多植物的花瓣。蜜蜂和蝴蝶看到的紫外线类似于飞机场的跑道，以便它们能够瞄准想吃的花蜜。 3、 有些蛇具有红外感应能力，以便“看见”猎物 蟒蛇、响尾蛇及其它种类蛇能够看到红外线，这意味着它们具有红外感应能力。这类蛇的颊窝（眼睛和鼻子之间）能够感应周围温度的变化，确定生物的准确位置，从而进行追踪和袭击。它们的红外视觉很精确，能够“看见”一米以内的猎物，并且能够以0.002℃的精确度感应到温度变化。 4、 有一些动物能够在晚上看到颜色 在黑暗环境下，人类眼睛中的视锥细胞会停止工作。这种情况下，只有视杆细胞工作，所以我们能够感应到阴影。但是，自然界中有一些物种依然能够在晚上看到颜色，例如壁虎。 相比于我们，壁虎在黑暗中对光线的敏感能力强350倍。这种能力对壁虎的生存很重要，因为它们的眼睛实际上没有视杆细胞，所以在进化过程中，其视锥细胞同时承担着视杆细胞的功能，且更敏感。 另外一种有此功能的生物是象天蛾（elephant hawkmoths），在白天它们以与蝴蝶一样的模式分辨花朵的颜色。 5、 皮皮虾：赶超所有动物，拥有最复杂的色彩视觉系统 在动物界，谁才是色彩视觉的王者呢？答案是：皮皮虾。 皮皮虾拥有16种视锥细胞，它们看到的颜色种类是人类的10倍之多，赶超地球上所有的动物。它们能够看到紫外线、红外线，甚至于偏振光。比外，它们的眼睛构造独特，能够独立转动，这意味着它们可以一次从两个不同的方向看到猎物和捕食者。 除了视觉系统惊人之外，皮皮虾的攻击速度也名列前茅。部分品种的身体下面甚至藏着一对儿能以60公里速度出击的“锤”。当皮皮虾攻击猎物时可以在10万分之一秒内将锤弹射出去，弹射的冲击力度最高能达到60kg，瞬间由摩擦产生的高温甚至能让周围的水冒出电火花。
